## FAQ

**Q: What if I can't post tomorrow?**  
A: Push to Wednesday. But consistency matters – pick a schedule and stick to it.

**Q: Should I post at 8 AM?**  
A: That's optimal for Australian professional audience on Tuesday/Thursday. Adjust based on your audience timezone.

**Q: Do I need all these hashtags?**  
A: Use 8-12. Mix broad + niche. The strategy doc lists which ones work best.

**Q: What if I get a comment asking for details?**  
A: That's a qualified lead. Reply with insight, then suggest a strategy call.

**Q: How long should I run this strategy?**  
A: Minimum 8 weeks to see real patterns. By week 4, you'll know what's working.

**Q: Can I repost the same post later?**  
A: Yes, but wait at least 3 weeks. People need time to forget it.

**Q: Do I need to create visuals for every post?**  
A: No – only for ~60% of posts. The other 40% are text-only for variety.

**Q: What if someone says this is a scam?**  
A: Don't engage defensively. Use the objection template in the checklist.
